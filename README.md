# Environmetus

Sitio web de consultoría ambiental enfocada en sostenibilidad y desarrollo responsable.

## Características

- Diseño responsive
- Secciones: Inicio, Quiénes Somos, Servicios, Proyectos, Contacto
- HTML/CSS puro (sin dependencias)

## Ver localmente

Abre `index.html` en tu navegador.

## Desplegar en GitHub Pages

1. Ve a Settings > Pages
2. Selecciona branch `main` y carpeta `/ (root)`
3. Guarda y espera unos minutos

Tu sitio estará en: `https://[tu-usuario].github.io/environmetus-website/`

## Personalizar

- Edita `index.html` para cambiar contenido
- Modifica `style.css` para ajustar colores y estilos
- Colores principales: `#2c5f2d` (verde principal)
